# Helper scripts

These scripts are optional convenience wrappers. They do not replace the analysis notebooks.

- `unpack_packaged_data.R` - unpacks raw object-level data archives that are tracked in compressed form.
- `check_inputs.R` - checks whether expected notebooks, metadata, data archives, and raw data patterns are present.
- `render_all.R` - attempts to render the main notebooks, skipping analyses whose required raw data or software are missing.
