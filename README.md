# TAD boundary replication analysis

Repository URL: <https://github.com/CBIIT/tad-boundary-replication-analysis>

This repository contains R, R Markdown, and Quarto analysis code supporting the manuscript **"Re-establishment of TAD boundary organization during DNA replication"**. The code focuses on high-throughput DNA FISH spot-distance analysis, including quantification of TAD boundary distances from object-level microscopy exports and cell-cycle staging from DAPI/EdU measurements.

## Contents

```text
.
├── analyses/
│   ├── figure3_cell_cycle_boundary_distance/

├── docs/

```

### Analysis folders

`analyses/figure3_cell_cycle_boundary_distance/` contains the Quarto notebook used for the FISH + EdU cell-cycle boundary-distance analysis. The notebook reads HiTIPS nuclei-level and spot-level CSV exports, calculates nearest-neighbor green/red FISH distances, normalizes DAPI intensity, assigns cell-cycle phase using DAPI and EdU, and plots boundary-distance distributions by cell-cycle phase.

`analyses/miniscreen_pilot/` contains R Markdown notebooks for the 210521 high-throughput FISH mini-screen workflow. The main notebook reads Columbus object-level exports, filters nuclei by FISH spot counts, calculates 2D and 3D green/red spot distances, and plots distributions by sgRNA target. This workflow represents the general FISH distance-analysis logic used across related analyses in the manuscript.

### Documentation

`docs/figure_code_map.md` maps manuscript figures and extended data tables to the code and data files provided in this release.

`docs/data_requirements.md` describes the expected input file patterns and key columns for the HiTIPS and Columbus object-level exports.

`docs/analysis_workflow.md` summarizes the FISH distance workflow and the cell-cycle staging procedure.

`docs/software_environment.md` summarizes the R versions and package versions recorded in the per-analysis `renv.lock` files.

## Data included in this repository

The repository includes small supporting tables and analysis metadata:

- Extended Data Tables 1-4 in `data/extended_tables/`.
- Plate-layout metadata for the included analysis notebooks.
- Rendered HTML reports and small output plots that were generated by the included notebooks.
- A compressed copy of the 210521 mini-screen Columbus object-level exports in `analyses/miniscreen_pilot/data_raw/miniscreen_columbus_exports.zip`.

Raw microscopy images are not stored in this code repository. Large raw object-level exports needed for complete reproduction of some analyses should be obtained from the associated manuscript data availability resources.

## Required data for the Figure 3 cell-cycle analysis

The Figure 3 notebook requires HiTIPS CSV exports in:

```text
analyses/figure3_cell_cycle_boundary_distance/data/
```

The notebook searches recursively for:

```text
*nuclei_information_well*.csv
*spots_locations_well*.csv
```

These large raw CSV exports are not tracked in this repository. The expected file patterns and columns are documented in `analyses/figure3_cell_cycle_boundary_distance/data/README.md` and `docs/data_requirements.md`.

## Unpacking the mini-screen data

The 210521 mini-screen Columbus exports are tracked as a compressed archive to keep the repository small and to avoid storing hundreds of individual raw export files.

To unpack them before rendering the mini-screen notebook:

```bash
Rscript scripts/unpack_packaged_data.R miniscreen
```

The mini-screen notebook also attempts to unpack this archive automatically if the expected data files are absent.

## Running the analyses

Each analysis folder keeps its own `renv.lock`. Restore the environment from the analysis folder being rendered.

### Figure 3 cell-cycle analysis

```bash
cd analyses/figure3_cell_cycle_boundary_distance
Rscript -e 'install.packages("renv"); renv::restore()'
quarto render 230918-ETB078-FISH-EdU-Opt3.qmd
```

This render requires the HiTIPS CSV exports described above.

### Mini-screen analysis

```bash
Rscript scripts/unpack_packaged_data.R miniscreen
cd analyses/miniscreen_pilot
Rscript -e 'install.packages("renv"); renv::restore()'
Rscript -e 'rmarkdown::render("210521-MiniScreen.Rmd")'
```

### Check expected inputs

From the repository root:

```bash
Rscript scripts/check_inputs.R
```

### Render available notebooks

From the repository root:

```bash
Rscript scripts/render_all.R
```

The render script skips the Figure 3 notebook when the required HiTIPS raw CSV exports are absent.

## Reproducibility scope

This repository is a transparent analysis-code release for the FISH distance workflows and supporting tables associated with the manuscript. It is not a raw image repository, and it does not contain code for every final figure-assembly step. In particular, raw microscopy images, large Figure 3 HiTIPS exports, and final graphical assembly files are managed outside this Git repository.
