## Practical interpretation

The repository is strongest as a transparent code release for the FISH-distance analysis workflow and the Figure 3 cell-cycle analysis. It is not yet a one-click reproduction of every final manuscript panel because several final data sources and figure-assembly steps were not included in the uploaded archives.
