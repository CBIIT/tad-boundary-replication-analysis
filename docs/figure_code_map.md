# Figure-to-code map

This map describes how the manuscript figures and extended data tables relate to files included in this repository. The repository is primarily an analysis-code release for DNA FISH distance quantification and cell-cycle staging, with selected small supporting tables.

## Main figures

| Manuscript item | Relevant repository files | Coverage in this repository | Notes |
|---|---|---|---|
| Figure 1: high-throughput screen identifies regulators of TAD boundary proximity | `analyses/miniscreen_pilot/210521-MiniScreen.Rmd`; `data/extended_tables/Extended Data Table 1.xlsx`; `analyses/figure3_cell_cycle_boundary_distance/funcs/plotting.R` | General workflow for reading high-throughput object-level FISH data, filtering cells by spot counts, calculating green/red spot distances, and plotting distributions. Extended Data Table 1 contains CRISPR screen gene z-scores and boundary-distance values. | The included mini-screen notebook represents the FISH distance-analysis workflow rather than every final panel-assembly step for Figure 1. |
| Figure 2: RPA complex modulates TAD boundary structure | `analyses/miniscreen_pilot/210521-MiniScreen.Rmd`; `data/extended_tables/Extended Data Table 2 siRNA Library Validation v2.xlsx`; `data/extended_tables/Extended Data Table 3 BAC Probes.xlsx` | The same FISH spot-distance workflow can be used for RPA subunit knockdown/validation analyses; tables provide validation-library and probe metadata. | Final manuscript panels may combine additional raw validation data and graphical assembly outside this code repository. |
| Figure 3: inhibition of RPA interrupts post-replication TAD boundary re-establishment | `analyses/figure3_cell_cycle_boundary_distance/230918-ETB078-FISH-EdU-Opt3.qmd`; `analyses/figure3_cell_cycle_boundary_distance/funcs/plotting.R` | Direct cell-cycle analysis notebook: reads HiTIPS nuclei/spot CSVs, calculates spot distances, normalizes DAPI by G1 peak using `mclust`, assigns DAPI/EdU cell-cycle phase, and plots boundary-distance distributions. | Rendering requires the large HiTIPS CSV exports described in `docs/data_requirements.md`. | Figure 4: RPA inhibition affects RAD21 loading and spatial distribution | `data/extended_tables/Extended Data Table 4 Sequencing tracks.xlsx` | The sequencing track table lists identifiers for tracks used in figure panels. | ChIP-qPCR plotting, MINFLUX processing, and final figure assembly are outside the included FISH-distance notebooks. |

## Practical interpretation

The repository provides reproducible analysis code for the FISH-distance workflows and the Figure 3 cell-cycle analysis. Some final manuscript panels combine these analyses with external data sources, raw image data, genome-browser-style tracks, or figure-assembly steps that are not stored in this code repository.
