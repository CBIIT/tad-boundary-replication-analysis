# Figure 3 cell-cycle TAD boundary distance analysis

This folder contains the Quarto analysis notebook for the FISH + EdU cell-cycle TAD boundary distance analysis.

## Main notebook

```text
230918-ETB078-FISH-EdU-Opt3.qmd
```

The notebook reads HiTIPS single-cell and spot-level CSV exports, calculates green/red FISH spot distances, assigns cell-cycle phase from DAPI and EdU intensity, and plots boundary-distance distributions by cell-cycle phase.

## Relationship to manuscript analyses

This is the primary included workflow for analyses of TAD boundary distance during DNA replication and cell-cycle progression. It is also the clearest example in this repository of DAPI/EdU cell-cycle staging from high-throughput microscopy exports.

## Analysis steps in the notebook

1. Load R packages and helper plotting functions.
2. Read the plate layout metadata.
3. Read HiTIPS nuclei-level data and calculate DAPI/EdU intensity summaries.
4. Read HiTIPS spot-level data and convert x/y/z coordinates to microns.
5. Filter cells based on spot count and nuclear quality.
6. Calculate pairwise 2D green/red spot distances with `SpatialTools::dist2`.
7. Retain the minimum green/red distance per red spot as the paired TAD boundary distance.
8. Fit a two-component Gaussian mixture to DAPI intensity per well to normalize the G1 peak.
9. Assign cell-cycle phases using DAPI and EdU thresholds.
10. Plot FISH distance distributions by cell-cycle phase.
