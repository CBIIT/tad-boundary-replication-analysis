# 210521 FISH distance mini-screen analysis

This folder contains the R Markdown analysis notebooks for the 210521 high-throughput DNA FISH mini-screen workflow.

## Main notebooks

```text
210521-MiniScreen.Rmd
Columbus_3D.Rmd
```

`210521-MiniScreen.Rmd` reads Columbus object-level data exports, filters nuclei, calculates 2D and 3D FISH spot distances, and plots distance distributions by sgRNA target.

`Columbus_3D.Rmd` is an earlier/general Columbus 3D distance workflow retained for provenance.

## Relationship to manuscript analyses

This folder documents the general high-throughput FISH spot-distance workflow used for TAD boundary measurements. It is most directly relevant to the screen and validation analyses that quantify green/red FISH probe separation across perturbations.

## Included inputs

The raw Columbus object-level exports are stored as a compressed archive:

```text
data_raw/miniscreen_columbus_exports.zip
```

Unpack the data before rendering:

```bash
cd analyses/miniscreen_pilot
Rscript ../../scripts/unpack_packaged_data.R miniscreen
```

The archive expands into `data/` and contains Columbus per-well files for nuclei, 488-channel spots, and 561-channel spots. The notebook expects files matching:

```text
data/*Nuclei Final[0].txt
data/*Spots 488[1].txt
data/*Spots 561[2].txt
```

Do not rename the Columbus export files unless the glob patterns in `210521-MiniScreen.Rmd` are also updated.

## Included outputs

The repository includes the rendered HTML report and plot PNGs generated by the notebook:

```text
210521-MiniScreen.html
output/*.png
```

## Run

```bash
cd analyses/miniscreen_pilot
Rscript -e 'install.packages("renv"); renv::restore()'
Rscript -e 'rmarkdown::render("210521-MiniScreen.Rmd")'
```

The notebook attempts to unpack `data_raw/miniscreen_columbus_exports.zip` automatically if the expected files are absent.

## Environment notes

`renv.lock` records the R and package versions for this analysis. Restoring the lockfile is recommended for exact reproduction because the notebook was written against older tidyverse versions.
