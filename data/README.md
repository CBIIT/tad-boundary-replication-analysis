# Data included in this repository

This repository includes only small tabular data that were present in the uploaded manuscript/source archives.

## Included

- `extended_tables/` - Extended Data Tables 1-4 from the manuscript package.
- `analyses/miniscreen_pilot/data/` - Columbus object-level exports included in the `210521-MiniScreen` archive.
- `analyses/figure3_cell_cycle_boundary_distance/output/per_well_measurements.csv` - per-well summary generated by the Figure 3 notebook and included in the uploaded archive.

## Not included

- Raw microscopy images.
- The large HiTIPS raw CSV `data/` folder for `230918-ETB078-FISH-EdU-Opt3`, which was intentionally removed before upload.
- Complete data/code for final Illustrator assembly, ChIP-qPCR, MINFLUX, and sequencing-track panels, unless represented in the supplied extended tables.

For public release, add links or accessions for the image/data repository that will host raw microscopy images and large exported analysis files.
