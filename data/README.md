# Data included in this repository

This repository includes small tabular data, analysis metadata, rendered reports, and packaged object-level exports that support the analysis code.

## Included

- `extended_tables/` - Extended Data Tables 1-4 from the manuscript.
- `analyses/miniscreen_pilot/data_raw/miniscreen_columbus_exports.zip` - packaged Columbus object-level exports for the 210521 mini-screen analysis.
- `analyses/figure3_cell_cycle_boundary_distance/output/per_well_measurements.csv` - per-well summary generated by the Figure 3 cell-cycle notebook.

## Not included

- Raw microscopy images.
- Large HiTIPS object-level CSV exports for the Figure 3 cell-cycle analysis.
- Final graphical assembly files for manuscript figures.

The expected file formats and input locations are documented in `docs/data_requirements.md` and in each analysis folder.
